# Society Security Solution

A web-based application for managing residents, staff, and visitors in a housing society.

## Features

*   **User Roles:** The application has three user roles: Admin, Supervisor, and Resident.
*   **Admin:** The admin can manage buildings, flats, residents, and staff.
*   **Supervisor:** The supervisor can manage visitors, regular vendors, and staff details. They can also look up flat details and check maintenance status.
*   **Resident:** The resident can view visitor details.
*   **Login:** A secure login system for all users.

## Technologies Used

*   **Frontend:** HTML, CSS
*   **Backend:** PHP
*   **Database:** MySQL

## Setup

1.  Clone the repository.
2.  Import the `society_security.sql` file into your MySQL database.
3.  Update the database credentials in `utils/db_conn.php`.
4.  Run the application on a local server.
